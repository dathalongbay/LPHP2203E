        2. document ready 
        đảm bảo đoạn code bên trong nó sẽ chỉ chạy khi trang đã load xong dom 
        ví dụ thay đổi 1 hình ảnh trong trang html nhưng khi trang chưa load xong img 

        $(document).ready();
        trong () của ready chúng ta sẽ truyền hàm vô danh vào đây 
        function() {}
        nếu cú pháp của document ready hoàn chỉnh sẽ là 
        $(document).ready(function() {
            // code sẽ chạy khi dom được load xong 
        });
