cú pháp của min-width
        @media screen and (min-width: 1080px) {
            /* đoạn css áp dụng màn hình từ 1080px trở lên
            1080 đến vô cực
            */
        }

        sử dụng max-width trong media screen 
        @media screen and (max-width: 1080px) {
            /* đoạn css áp dụng màn hình từ 1080px trở xuống 
            1080px trở xuống 0px 
            0px -> 1080px 
            */
        }
    
        trong file css dùng nhiều media min-width thì chúng ta 
        viết tăng dần 
        còn nếu dùng nhiều media max-width thì chúng ta viết giảm dần
